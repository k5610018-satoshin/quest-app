はじめに — 本当に同期が必要ですか？

💡 ご注意: このアプリはクラウド同期なしでも全機能が動きます。 以下に該当しない人は、設定しなくて OK です：
  • 家のPCと学校のPCで同じデータを見たい
  • スマホでも記録したい
  • 自動でクラウドにバックアップを取りたい
単独PCで使うなら、「⚙ 設定 → 📥 JSONエクスポート」でバックアップを取るだけで十分です。

セットアップに必要なもの

  • ✅ Google アカウント（Gmail があればOK）
  • ✅ ブラウザ（Chrome/Edge/Firefox/Safari）
  • ✅ 5〜7分の時間

このアシスタントが代わりにやってくれること

  • 📋 GASコードをワンタッチでクリップボードにコピー
  • 🔗 必要なページを自動で新しいタブで開く
  • ✅ 入力された URL と API_KEY を、最後にアプリに自動反映

準備ができたら、下の 「次へ →」 をクリックして始めましょう。

ステップ 1: 新しい Google スプレッドシートを作る

下のボタンをクリックすると、Googleスプレッドシートの新規作成画面が新しいタブで開きます。

📄 新しいスプレッドシートを開く

新しいタブで開いたら、以下の操作をしてください:

  1. 左上の「無題のスプレッドシート」をクリックして、ファイル名を変更
    おすすめのファイル名: 担任記録-同期データ
    ※ なんでも OK です。後で見てわかれば。
  2. 上部メニューから 「拡張機能」 をクリック
  3. 表示されるメニューから 「Apps Script」 をクリック
    ファイル 編集 表示 挿入 表示形式 データ ツール 拡張機能 ▼
    ↓ クリックすると ↓
    アドオン Apps Script ←ここ マクロ
    「拡張機能」メニューを開いた状態
  4. 新しいタブで Apps Script のエディタ が開きます。
    (URL が script.google.com/... で始まるページ)
✅ ここまでできたら下の「次へ」をクリック

ステップ 2: Apps Scriptにコードを貼り付ける

下の 「📋 コードをクリップボードにコピー」 ボタンを押してください。

次に、Apps Script のタブに戻って:

  1. エディタに最初から書いてあるコード（function myFunction() {} など）を 全部選択して削除
    削除のコツ: エディタ内をクリック → Ctrl+A（全選択） → Delete
  2. 空になったエディタに 先ほどコピーしたコードを貼り付け
    貼り付け: Ctrl+V
  3. 左上の 💾 保存ボタン をクリック（または Ctrl+S）
    ▶ 実行 ⚙ デバッグ 💾 保存 ←ここ
    エディタ上部のボタン
  4. 左上のプロジェクト名（無題のプロジェクト）が表示されたら、わかりやすい名前に変更
    おすすめ: 担任記録 同期
✅ コードが保存されたら下の「次へ」をクリック

ステップ 3: quickSetup を実行して API_KEY を生成

Apps Script のタブで以下の操作をしてください:

  1. エディタ上部の関数選択プルダウン（関数を選択 や doPost と表示されている部分）を クリック
    ▶ 実行 ⚙ デバッグ doPost ▼ ←ここをクリック 📋 ログ
    エディタ上部の関数選択プルダウン
  2. 表示されるリストから 「quickSetup」 を選択
  3. 左にある 「▶ 実行」 ボタンをクリック
  4. 初回は 権限の承認ダイアログ が出ます:
    1. 「権限を確認」 をクリック
    2. 使うGoogleアカウントを選択
    3. 「Google ではこのアプリを確認していません」と出たら、左下の 「詳細」 をクリック
    4. 下の方の 「（プロジェクト名）に移動（安全ではないページ）」 をクリック
      ※ 自分自身が書いたスクリプトなので問題ありません
    5. 「許可」 をクリック
  5. 実行が成功すると、画面に 「✅ セットアップ完了！」 ダイアログが出ます。
    そこに表示される API_KEY（16文字のランダム英数字） をコピー！
    ✅ セットアップ完了！

    ━━━━━━━━━━━━━━━━━━━━━
    ...
    ↑ こんな感じで API_KEY が表示される
✅ API_KEY を上の欄に貼り付けたら下の「次へ」をクリック

ステップ 4: Web アプリとしてデプロイ

続けて Apps Script のタブで:

  1. 右上の 「デプロイ」 ボタンをクリック
    🔍 デプロイ ▼ ←ここ ⚙
    エディタ右上のボタン
  2. 表示されるメニューから 「新しいデプロイ」 をクリック
  3. 左上の ⚙ アイコン をクリックして、「ウェブアプリ」 を選択
    ⚙ ←クリック → アドオン ライブラリ ウェブアプリ ←選択 実行可能API
  4. 以下の設定にする:
    説明:v1 （何でも OK）
    次のユーザーとして実行:自分（あなたのアカウント）
    アクセスできるユーザー:全員
    ⚠ 重要: 「アクセスできるユーザー」を 「全員」 にしてください。
    「自分のみ」だと別PC・スマホから同期できません。
    API_KEY が無いと書き込めない仕組みなので、安全です。
  5. 「デプロイ」 をクリック
  6. 少し待つと、「ウェブアプリのURL」が表示されます。
    https://script.google.com/macros/s/.../exec 形式。
    「コピー」 ボタンで URL をコピー。
✅ URL を上の欄に貼り付けたら下の「次へ」をクリック

ステップ 5: 担任記録アプリに反映

入力された設定を確認:

WebアプリURL:
API_KEY:

選択肢A: 自動でアプリに設定（おすすめ）

下のボタンを押すと、ブラウザに保存された設定を自動で更新します。

選択肢B: 手動で設定する場合

  1. 担任記録アプリ（index.html）を開く
  2. 「⚙ 設定」 タブをクリック
  3. 「クラウド同期 (GAS)」 カードを探す
  4. WebアプリURLとAPI_KEYを上記の値で入力
  5. 「同期を有効にする」 を ON
  6. 「今すぐ同期」 をクリック
🎉 お疲れ様でした！ これで複数PC・スマホで自動同期できる状態になりました。
記録すると数秒〜5分以内にクラウドへ送信され、別端末では「今すぐ同期」で取得できます。
🚀 さらに便利に: ?autosync=1 ブックマーク
起動と同時に「全層完全同期(L1↔L2↔L3)」を自動実行する起動オプションがあります。
使い方:
  1. 下のリンクを右クリック → 「ブックマークに追加」
  2. 毎回このブックマークから開けば、最新クラウドデータが起動5秒後に自動反映
📌 担任記録アプリ (起動時自動同期)
※ 5秒待ってから fullSyncAllLayers() を実行 → 結果はトーストとタイトル(✅/⚠) で確認できます。
※ 編集中フラグは自動解除されます (バックグラウンドタブで止まらないように)。
🆘 もしデータが消えたら
  • 緊急復元ボタン: 設定タブ → クラウド同期 → 「🆘 クラウドから緊急復元」 (赤ボタン)
  • diagnostic.html: アプリと同じChromeで開くと L1/L2/L3 の件数差を一覧表示
  • auto-sync.html: ?autosync=1 と同じ処理を独立ページで実行 (SWキャッシュ非依存)
  • recovery.html: 3-step手動復旧ツール
