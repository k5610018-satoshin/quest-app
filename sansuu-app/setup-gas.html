セットアップ手順

  1. GASプロジェクト gas-sansuu-bridge をデプロイ（gas/ ディレクトリ参照）
  2. Web App として「全員（匿名含む）」アクセス許可で公開 → エンドポイントURLを取得
  3. Apps Script Properties で SUPABASE_URL / SUPABASE_SERVICE_KEY / GAS_BRIDGE_API_KEY / SHEET_ID を設定
  4. このページに エンドポイント と API Key を入力 → 保存
  5. 接続テストでOKならアプリで自動同期される
